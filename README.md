# üõ† Troubleshooting
Model Loading Issues:

```bash
# Clear Hugging Face cache
rm -r ~/.cache/huggingface

CUDA Out of Memory:

# Reduce batch size in code (look for self.batch_size)
# Use smaller model
python ... --model-size 600M

Encoding Problems:

# Convert to UTF-8
iconv -f ORIGINAL_ENCODING -t UTF-8 input.srt > fixed.srt

```
